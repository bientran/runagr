# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`
- `PATCH /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Routes

- `GET /api/routes`
- `GET /api/routes/:id`
- `POST /api/routes`
- `PATCH /api/routes/:id`
- `DELETE /api/routes/:id`

### Activities

- `GET /api/activities`
- `GET /api/activities/:id`
- `POST /api/activities`
- `PATCH /api/activities/:id`
- `DELETE /api/activities/:id`
